# Bountiful Delve Tab

Adds a dedicated Bountiful Delves tab to the World Map, giving you a quick at-a-glance list of all Midnight delves and their current bountiful status — without having to hunt through each zone map individually.

---

## Features

- **All Midnight delves in one place** — a clean dedicated list so you're not hunting for delve icons buried under world quests and other map clutter
- **Live bountiful detection** — pulls directly from the game's own map pin data as you browse zones, no separate tracking required
- **Smart sorting** — bountiful delves float to the top in gold, regular delves listed below, unvisited zones shown as unknown
- **Click to set waypoint** — click any delve in the list to set it as your active waypoint
- **Session caching** — bountiful status is saved as you visit zones, so you only need to browse each map once per login

---

## How It Works

Open your World Map and click the **Bountiful Delves** tab on the right side (the glowing golden archway icon). The list shows all delves with their current status. Status is populated automatically as you navigate zone maps — browse through Quel'Thalas zones and the list will fill in.
